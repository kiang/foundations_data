類型

法人代表

設立登記日期

主事務所
新北市板橋區中山路1段265巷11號1樓 
目的
為促使社會深入瞭解視障者在各種職業、生活與運動中，綜合運用視覺以外的相關能力（聽覺、觸覺、嗅覺、味覺、本體覺、空間覺、障礙覺）發展出獨特的非視覺文化，以訓練視覺障礙者運動與生活的專門技術，在協助視障者提昇適應社會能力外，讓社會大眾透過活動參與使用非視覺能力操作實務，理解非視覺能力如何在影像環境達成任務與目標，進而改變人們對視障者操作能力的觀感，從而具備接納視障者生活能力與職業能力的友善態度，以積極正面行動支持視障者，增進企業提供多元職場接納視障者，促進社會環境自然支持視障者生活的宗旨。 
捐助方法
入會費、常年會費、永久會費、事業費、會員捐款、委託收益、基金及其孳息、其他收入
