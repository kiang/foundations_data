目的
本會以運用社會資源，結合民間力量，舉辦教育事務為宗旨，依有關法令辦理下列業務：（一）學生獎助金之設置。（二）學生及教職員員工（含退休人員）之急難救助與照顧。（三）充實圖書、行政、教學相關設施及設備。（四）校友、家長及捐款人士之聯繫服務。（五）捐款人指定之項目。（六）其他有關教育、公益及學術性活動之舉辦與補助。 
捐助方法
本會以本校文教基金籌備會新台幣二百萬元為創立基金。俟後運作經費來源如下：（一）基金孳息。（二）民間捐款。（三）其他收入。
