目的
1.協助政府有關機關推行環境保護活動。2.辦理各項環境監測、保護等研討會活動。3.宣導環境保護知識，推廣環保教育、提高民眾環保意識等各項活動。4.宣導健全生態保育工作，防治公害發活動。5.辦理落實美化生活環境，促進安和樂利的社會。6.辦理其他與生態、環境教育有關之事項。7.辦理環境保護活動。8.環境保護志願服務之辦理及推廣。9.贊助研究發明工作。10.其他相關環境保護社會福利項目。11.接受主管機關指導辦理事項。12.獎助學金之捐助。 
捐助方法
基益企業（股）公司捐助伍佰萬元整，益通光能科技（股）公司捐助伍佰萬元整共計壹仟萬元整。
