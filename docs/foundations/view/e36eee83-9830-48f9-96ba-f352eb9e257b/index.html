目的
由於第四次工業革命附隨著人工智慧的社會再現力，遽增婦女在家庭與社會角色的重要性，本會以婦女觀點出發，強化婦女於家庭中之角色功能，建構美麗新生活，進而創造包容且多元的社會。本會宗旨為： 1. 結合關注性別議題之婦女相互扶持，喚起女性意識，促進婦女成長以促進婦女福利。 2. 爭取婦女在法律和工作上之平等。 3. 支援不幸婦女，發展婦女培力方案，照顧處於不同生命週期、族群、處境的婦女需求。
