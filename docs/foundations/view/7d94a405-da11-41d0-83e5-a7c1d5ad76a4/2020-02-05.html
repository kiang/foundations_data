目的
本會為依法設立，非以營利為目的之社會團體；結合農產業、行政、學術界之力量，以研究農業經營管理之理論與實務，提升管理科學與技術，促進農業永續發展為宗旨。本會所稱農產業，係涵蓋農作、森林、水產、畜牧等動植物產銷及加工、服務之事業。
