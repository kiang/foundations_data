基本資料

 

類型
社團 
法人代表

設立登記日期
2018-07-04 
主事務所
雲林縣虎尾鎮埒內里僑豐街7巷5號 
目的
本會主要以推動社區教育學習、環境生態關懷為理念，並且從事教育研習、地方藝文展覽與活動規畫、社區營造、文化旅遊等工作，促進地文化深根、提升社區教育與藝文品質並由雲林在地青年去把創新、創意、創業3個理念，開發社區學習型產業資源、深耕社區文化之總體營造、監督自然生態環境資源、守護地方文化資產保存與維護、人文關懷結合民間慈善社團之力量，配和政府社會福利之推行，共同開創地方教育推展、社會關懷事業及地方總體營造等議題倡導為宗旨。 
捐助方法
1.入會費。2.常年會費。3.事業費。4.會員捐款。5.委託收益。6.基金及其孳息。7.其他收入。
