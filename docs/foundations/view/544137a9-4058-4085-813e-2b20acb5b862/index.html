(一)為保存中華民國固有優良傳統文化，闡揚天人合一靈性修煉，達到通徹天地之境界。復興古聖先賢的道義，結合專家學者，悟明其奧義，以博愛、慈悲、忠恕、清真、感應、無為藉以代天宣化，薰陶人心，堅定心法道，並以四維(禮義廉恥)八德(忠孝仁愛信義和平)結合崇信道學人士，共同闡揚大道清虛之理，進而揖止人性邪惡，提昇中華精神教育，厚植文化根基，促進人類和平，達到大道之行，天下為公，世界大同之宗旨。 （二）以慈（慈悲、博愛）、儉（惜福、環保）、善（善言、行、念）、讓（禮讓、圓融）為號召傳播社會。
