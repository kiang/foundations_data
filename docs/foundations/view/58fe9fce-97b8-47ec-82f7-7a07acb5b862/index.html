目的
本會為依法設立、非以營利為目的之社會團體，為協助政府推行政令，對身心障礙者提供就業、福利、技能訓練、創業、心靈、運動休閒、醫療、法律輔導之必要扶助，並結合社會各界樂善好施、熱心公益人士、團體，共同來扶助身心障礙者，讓他們的身心平衡發展，且能在互助扶持之下，發揮別人能、我也能之精神，期能達到互助互勵、創造未來新希望，且能服務社會為目標，進而融入社會、回饋社會為宗旨。 
捐助方法
常年費、團體費、贊助費、顧問費、事業費、政府補助及各界捐助等方法
