花蓮縣新城鄉身障弱勢及原住民關懷服務協會 
目的
本會為依法設立，非以營利為目的之社會團體，以協助殘障者、中（低）收入者、原（新）住民、老人、發揮自立精神、謀求社會資源；整合福利，並促進社會接納本會所輔導對象，建立親職、家庭教育、第二專長及故事媽媽與會員子女課後輔導活動推展及轉介服務為宗旨及發展地方、社區自有產業以協助本會協助對象，改善生活、就業、培育第二專長計劃，輔導進入常態職場並增進相關就業產福祉，並延續發展文化產業、農業及林業、生態教育及志工營對等種子老師培育計劃為宗旨。 
捐助方法
個人會員、團體會員、贊助會員、名譽會員、入會費、常年會費、會員捐助、社會捐助、基金及其孳息、委託收益及如章程第29條之其他收入。
