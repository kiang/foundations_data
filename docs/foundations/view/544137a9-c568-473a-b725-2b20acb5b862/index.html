目的
以勞資雙方權利義務與各項權益促進、協助勞資雙方對勞動法令之研究與瞭解及所有弱勢族群與人民社會福利之維護促進，並從事相關法規之宣導與專案工作為宗旨。一、促進勞資雙方之權益爭取。二、勞資中介團體協助勞資協調調解仲裁。三、提供實際專業服務，以保障促進弱勢族群之各項權益，例如提供受虐外籍人士、新住民、本國人民庇護及相關法律協助，並承辦政府委外專案。四、積極辦理政府委託有關勞資權益、人力資源管理、社會福利與其他項目之教育訓練課程及其它政府委辦專案。五、協助政府推行及宣導勞動政策及法令，並承辦相關政府委外專案。六、協助政府推行及宣導社會福利保障促進、政策及相關法令宣導，並承辦相關政府委外專案。七、其他勞資權益與社會福利相關服務事宜。八、政府其他委外專案承辦。 
捐助方法
入會費、捐款、會員服務費、事業費、委託收益、基金及其孳息、其他收入。
