目的
促進台灣地區腦膜炎教育訓練及健康組織，藉以提高國人對腦膜炎議題的關注。一、邀請專業醫療人員對公眾進行腦膜炎等多種病因的教育。二、讓人民瞭解腦膜炎、菌血症對生活所造成的影響。三、對上述疾病的倖存者及家屬提供幫忙。四、促請政府撥款衛生相關部門增加疫苗接種預算，並為腦膜炎研究提供資金。五、積極參與並加入國際腦膜炎組織（CoMO)，以提高台灣腦膜炎協會的能見度。
