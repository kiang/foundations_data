目的
本會以追求「醫學美容醫療品質安全與國人健康」為主要目的、自律導正國內醫美市場風氣，從民間機構做起，替民眾在醫學美容第一線把關為使命，提供檢驗合格之耗材針劑，藉由本會匯集更多相關從事醫學美容相關領域之醫療器材產品之供應商、業者、醫療機構、護理人員及醫師之專業，共同推動醫學美容品質保障之任務，確保醫學美容醫療環境使用產品之安全、並教育社會大眾對醫學美容醫療安全的正確認識，以促進國民醫學美容醫療安全福祉為宗旨，最終期許能喚起社會大眾對醫學美容醫療品質之重視，並加深消費者信賴，提升我國國際醫療觀光能見度。
