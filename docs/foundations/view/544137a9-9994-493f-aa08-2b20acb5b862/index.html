主事務所
台南市安平區安平路97號之10 
目的
本會以為紀念安平耆老何世忠先生對安平地區文化發展之貢獻，延獻其理念，扶助安平地區發展文化藝術工作，奬助有關團體，培育人才，擴展文化藝術資源，達到推廣社會教育之目的為宗旨，依有關法令規定辦理下列業務：㈠辦理有關安平藝術資源之探尋、蒐集、整理與保存工作。㈡推動保存安平具有歷史價值空間、文物之探索、紀錄、整理、發表等相關工作。㈢出版有關安平文化藝術活動之論文、報告及期刊、適時規劃學術研討、民俗藝文活動。㈣奬助安平地區之文化藝術工作者、團體、學校社團，其展覽、表演及論述作品之發表活動。㈤協助安平地區兒童、青少年及弱勢家庭文化藝術活動。㈥舉辦國內外有關文化藝術交流訪問活動。㈦其他符合本會設立宗旨之相關公益性文化藝術活動。 
捐助方法
本會設立基金共新台幣陸佰萬元整，由何魏玉蘭女士、何應冠先生、何應騰先生、應大實業有限公司、應億企業有限公司及建成實業廠捐助。
