目的
為弘揚顯密圓融之佛陀教法，並落實本派（嘎檔派）所依循之藏傳佛教教典精髓，故將源自印度阿帝夏尊者所傳之菩提道燈論及以此為基礎的菩提道次第廣論發揚光大。並為廣大培養佛法人才，化育人心，修十善業培養社會良好風氣，故成立中華國際種敦巴佛教會。顯的部分以菩提道燈論為主要推廣之教法內容。密的部分則以具德金剛令怖尊佛、十六圓點為主要修法儀軌。本學會以藏傳佛教種敦巴仁波切之名而設，乃因種敦巴尊者傳承自阿帝夏大尊主所創之嘎檔派之所有教法，本學會以實踐嘎檔派之修行次第為宗旨。
