目的
本會以關懷家庭關係及兒童、青少年之生命重視、倡導健全家庭關係、推動社區活化發展、宣揚基督真愛信仰為其宗旨。本會之任務如下：一、家庭（含弱勢、單親、低收入戶）關懷輔導事工。二、兒童及青少年（含行為偏差、品格）關懷輔導事工。三、正常兩性關係平衡發展輔導與心理諮商。四、健康家庭教育與社會責任之全人發展信念傳遞。五、一般心理諮商協談事工。六、銀髮族陪伴關顧事工。七、社區活化總體營造輔導事工。
