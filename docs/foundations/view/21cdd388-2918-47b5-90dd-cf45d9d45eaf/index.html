基本資料

 

類型
財團 
法人代表
董事長丁冠介 
設立登記日期
2018-10-23 
主事務所
苗栗縣卓蘭鎮上新里食水坑68號 
目的
一、關於老人、婦女、兒童及少年、身心障礙、弱勢家庭等社會福利服務及其身心關懷等相關公益事項。二、關於協助社會弱勢者及清寒家庭需求事項。三、關於提供社會急難救助及災害救助等事項。四、關於公益扶助事項。五、關於促進社會福祉及全人發展事項。六、關於推動社會慈善公益中心業務事項。七、關於主管機關指導辦理事項。八、辦理其他符合本會設立目的之相關公益性社會福利事務。 
捐助方法
本會由丁冠介等捐助成立，捐助金額為新臺幣1,000萬元。合計基金共新臺幣1,000萬元整。前項基金得由捐助人或其他個人、團體繼續捐贈補充之。
