主事務所
新竹縣竹東鎮二重里3鄰明星路227巷13號 
目的
（一）推動辦理轄內社區信仰中心「福德正神」之各項文化，客家大戲之推廣與傳承（酬神平安戲）。 （二）結合社區發展推動辦理有關幼兒教育及成長之活動。 （三）推動爭取距離現有社區活動中心暨投票所偏遠之較大型社區，設立小型社區活動中心供年長民眾做為 休憩場所兼投票所以提高老年長輩之投票率。 （四）辦理老人、婦幼各項福利之爭取，並協助弱勢家庭爭取照顧。 （五）協助辦理各項公益活動及社會福利事項。 （六）推動社區團體舞蹈及歌唱研習班等健康活動。 
捐助方法
入會費、常年會費、事業費、會員捐款、委託收益、基金及其孳息、其他收入。
