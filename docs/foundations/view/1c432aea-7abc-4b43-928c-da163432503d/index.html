目的
本會為依法設立、非以營利為目的之社會團體。宗旨如下：本協會以宣揚基督真理，並以實踐信望愛之精神投入全人關懷、身心靈教化及社區服務工作，以造福人群，特成立宜蘭縣榮耀糧倉協會。本會之任務：一、宣揚基督教義、廣傳福音、啟迪人心、心靈改革。二、辦理各項社會福利服務活動及補助並幫助貧困孤弱者。三、全人、家庭、社區的輔導和關懷服務及教育訓練。四、訓練教會神職人員，協助本會辦理啟迪人心、心靈改革的人才。五、其他符合本會宗旨之事項。 
捐助方法
一、會員捐款。二、其他教會、福音機構及其他個人或團體之捐款。三、基金及其孳息。四、舉辦募款活動。五、其他收入。
