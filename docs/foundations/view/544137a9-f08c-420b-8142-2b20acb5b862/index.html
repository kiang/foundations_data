主事務所
雲林縣莿桐鄉莿桐村南安路42之2號1樓 
目的
致力整合各界產官學相關多方文化資訊教育資源，透過多元的文化資訊教育發展讓民眾能更加了解，並運用社會資源幫助民眾提昇民眾對於多方面的多元文化資訊教育資訊的了解，以促進文化、資訊、教育為目的。 
捐助方法
一、入會費。二、常年會費。三、會員捐款或社會民眾捐款。四、委託收益。五、基金及其孳息。六、政府機關之補助。七、事業費。八、其他收入。
