目的
以訓練品質管理交流為宗旨。一、促進兩岸暨國際教育訓練人才之交流，共享知識與師資資源，提升訓練能量。二、促進公民營企業、機構或社會團體，重視訓練品質之提升與管理。三、提供政府有價值之參考資訊，協助政府更有效提供資源。四、提供更多失業訓練資源，提升國內就業市場之發展。
