目的
一、深入校園、社區及一般工作場所作愛滋和性教育宣導和舉辦相關活動。 二、藉刊物、講座，提供社會大眾服務之新知、法令、規章、知識、技術訓練、諮詢服務及心理協談。 三、辦愛滋相關講座、支持、聯誼暨成長團體，協助青少年愛滋感染者建立有效的支持平台，並透過專業人員與同儕員的陪伴，促進感染者生活品質提升。 四、辦理志工招募培訓和助人者工作坊，提升助人工作者的專業知能，以提供更適切的協助；對於會內的服務人員，提供定期的個別督導與與團體督導，以了解其服務提供的適切度，並給予心理支持。 五、接受政府委託辦理社會福利相關計畫。 六、舉辦其他符合本會宗旨之公益活動。
