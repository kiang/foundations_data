目的
本會為依法設立、非以營利為目的之社會團體，以維護臺灣天然攀岩活動場域之安全和生態保育，提升戶外攀岩專業知識與技術，推廣攀岩者安危自負精神，並促進臺灣戶外攀岩活動之永續發展為宗旨。 
捐助方法
1.入會費、2.常年會費、3.事業費、4.會員捐款、5.委託收益、6.基金及其孳息、7.其他收入。
