目的
本會宗旨：一、為發展我國免稅零售產業，促進產業共存共榮，凝聚免稅零售產業暨觀光產業，接軌全球免稅零售產業平台。二、建立免稅零售產業及其供應商與營運商之間的密切合作關係，並代表產業與協會成員為產業發展及共同利益，向各國政府與國際組織爭取業務支持，並舉辦動態發展論壇。三、提昇國際旅客服務品質，確保免稅零售及觀光產業永續發展。協調免稅產業暨相關產業關係，精進產業水準，增進共同利益。促進國際交流合作，敦睦國際友誼。四、協助政府政令推行與建議法令修正，促進國內免稅零售及觀光產業繁榮升級。五、其他有關免稅零售產業發展相關事項。 
捐助方法
入會費、常年會費、事業費、會員捐款、委託收益、基金及其孳息、其他收入。
