目的
一、配合政府的措施，爭取弱勢者該享有的福利，提升長者的生存權益，落實老人的福利推展，並提供保健、輔具申請等規劃性服務。二、有效推展老人的健康服務，維護長者的生活尊嚴，配合長輩的生活協助服務。三、日間照顧與法律咨詢等服務。四、推動有效的照顧服務政策，結合當地社區的有利資源，落實長期維護並達到被照顧者的需求。五、退休準備回歸自然的休閒服務資訊供及轉介服務，其他相關之社區式服務。六、照顧弱勢開發需要的人力資源，培訓專業的輔導員。 
捐助方法
一、入會費：個人會員新台幣壹仟元，團體會員新台幣壹仟伍佰元，於會員入會時繳納。二、常年會費：個人會員新台幣伍佰元，團體會員新台幣壹仟元。三、事業費。四、會員捐款。五、基金及其孳息。
