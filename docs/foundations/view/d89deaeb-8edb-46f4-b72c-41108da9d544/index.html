目的
本會以辦理研究國內外重大議題，並促進國際城市交流，建立學術界與產業界溝通平台，及提供政策評估與建言為目的。受益範圍以桃園市為執行業務區域。 本會依照有關法令之規定，並視實際需要及財力狀況辦理下列各項目的事業： 一、研究分析國家學術、教育、文化、整體發展等重大政策議題，提供政策評估與建議。二、整合產官學資源，為落實學術界、文化界及產業界跨領域合作。三、關於其他獎助及從事學術研究和優秀人才培育之各類事項。四、關於贊助其他文教公益團體活動事項。五、接受主管機關指導辦理事項。六、其他符合本會宗旨之相關公益性文教事項。
