目的
為發揚中華儒道，實踐孔子「志於道，據於德，依於仁，游於藝。」之涵養，本協會推展有助於道、德、仁、藝發展之社會公益活動，期望將中華儒道落實在社會各層面之生活當中，以重振中華固有之優良文化傳統，移風易俗，促進社會祥和，冀世界之大同。 
捐助方法
入會費、常年會費、事業費、會員捐款、委託收益、基金及孳息、其他收入。
